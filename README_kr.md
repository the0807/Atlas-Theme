# Atlas Theme for Oh My Posh

|![img1](img/img1.png)|![img2](img/img2.png)|
|:---:|:---:|

깔끔하고 심플한 [Oh My Posh](https://ohmyposh.dev/) 테마로, Dracula+ 색상 조합과 D2Coding Ligature Nerd Font를 추천합니다.

## 📸 미리보기

이 테마는 다음 정보를 표시합니다:
- ⚡️ Root 권한 표시
- 👤 사용자 이름 및 명령 실행 상태
- 📁 현재 경로
- 🌿 Git 브랜치 및 상태
- 🐍 Python 가상환경 (활성화 시)
- ☸️ Kubernetes 컨텍스트 및 네임스페이스 (설정 시)
